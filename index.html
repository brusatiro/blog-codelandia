Comece por aqui: HTML

HTML significa Hypertext Markup Language. Ele não é uma linguagem de programação, mas sim de marcação, isto significa que ele vai apenas estruturar o conteúdo das... suas páginas ou aplicações. Vamos pensar no corpo humano, nele o HTML seria apenas a estrutura esquelética, e para o corpo ser visualmente bonito e realmente funcionar iremos precisar do CSS e do JavaScript.

Para que serve o CSS

O CSS é chamado de linguagem Cascading Style Sheet e é ele quem vai estilizar e deixar a sua estrutura HTML visualmente mais bonita. Com ele você pode organizar e decorar todo o layout da sua... página, alterando cores e tamanhos, estabelecendo posições diferentes para os elementos, ajustando imagens, etc. O CSS seria a parte de fora do corpo humano, o que nós enxergamos ao olhar para uma pessoa, como o rosto e as roupas.

Onde entra o JavaScript

Vamos falar sobre o JavaScript, ou simplesmente JS, agora sim estamos tratando de uma linguagem de programação! O JS é usado principalmente para o desenvolvimento... web e desenvolvimento de software, é ele quem vai trazer funcionalidade e interação com o usuário à sua página ou aplicação. Você pode imaginá-lo como a musculutarua ou até mesmo a alma do corpo humano, pois é ele quem traz vida, movimento e ação para a sua criação web. O JavaScript é uma das linguagens mais populares do mundo, pois ele pode atuar não só na programação front-end (parte visual), mas também no back-end (parte interna de uma aplicação).

Bônus: Frameworks

Ah, os frameworks! Eu não havia mencionado até então mas eles também são extremamente importantes. Um framework é um conjunto de códigos que auxilia o programador no desenvolvimento de... sites e aplicações, basicamente ele traz estruturas de códigos prontas para serem aplicadas em uma determinada funcionalidade do seu projeto. Existem diversos frameworks, como Angular, Vue e Bootstrap, mas antes de aprender mais sobre eles, é bom ter uma boa base de conhecimento nas linguagens citadas acima.
